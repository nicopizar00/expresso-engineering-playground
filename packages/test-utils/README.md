# @travel-playground/test-utils

Shared test helpers consumable from every test layer (unit, integration,
contract, e2e).

Planned contents:

- **Factories** for domain entities (trips, bookings, orders, users).
- **Builders** for OpenAPI / event payloads.
- **Test doubles** — in-memory repositories used by integration tests.
- **Time / clock** helpers for deterministic tests.

Today this package only reserves the workspace.
